Solutions Architect specializing in cloud-based contact center solutions on AWS. I work at **In2clouds**, one of the first AWS partners to implement Amazon Connect in Latin America, where I lead the technical design of CX solutions integrating AI, automation, and conversational experiences.

## Featured Project

### 🎧 [Amazon Connect Lab](https://github.com/GabrielFMarinho/amazon-connect-lab)

A hands-on challenge simulating a real contact center implementation for a fictional company called **Conecta Company**. Built for the AWS re/Start bootcamp community.

- Site publishing on **Amazon S3**
- Instance setup and widget configuration on **Amazon Connect**
- Conversational menu built with **Amazon Lex V2**
- End-to-end testing with **Historical Metrics** and **Contact Search**

> The challenge is designed so students receive a business requirements diagram and must design the solution themselves — no step-by-step instructions.
